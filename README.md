# Abiturab Vora - Portfolio

A modern, interactive portfolio website showcasing my skills as a Java Full Stack Developer.

## Project Structure

```
Portfolio/
├── index.html              # Single HTML file with all content
├── css/
│   ├── base.css           # Variables & resets
│   ├── cursor.css         # Custom cursor styles
│   ├── navigation.css     # Navigation bar styles
│   ├── hero.css           # Hero section & ticker
│   ├── about.css          # About section & stats
│   ├── skills.css         # Skills section
│   ├── resume.css         # Resume & experience section
│   ├── contact.css        # Contact section & forms
│   ├── footer.css         # Footer styles
│   ├── animations.css     # Keyframes & animations
│   └── responsive.css     # Mobile responsive styles
├── js/
│   ├── cursor.js          # Custom cursor logic
│   ├── animations.js      # Scroll reveals & bars
│   ├── form.js            # Contact form handling
│   └── navigation.js      # Active nav state
├── Resume.pdf             # Your resume for download
└── README.md              # This file
```

## Features

✨ **Custom Cursor System** - Interactive cursor that adapts to background colors
🎨 **Modern Design** - Clean, professional layout with vibrant color palette
📱 **Fully Responsive** - Looks great on all devices
🚀 **Smooth Animations** - Fade-up animations and scroll reveals
⌨️ **Navigation** - Fixed nav with active section highlighting
📋 **Sections Included:**
  - Hero (Introduction)
  - About (Personal info & stats)
  - Skills (With animated progress bars)
  - Resume (Experience & Tech Stack)
  - Contact (Links & contact form)
  - Footer

## Modular Architecture

### CSS Structure (11 Files)
- **base.css** - CSS variables, resets, body styles
- **cursor.css** - Custom cursor dot and ring
- **navigation.css** - Fixed nav bar with underline animation
- **hero.css** - Hero blocks, big name, ticker
- **about.css** - About grid, text, stat cards
- **skills.css** - Skills grid, cards, progress bars
- **resume.css** - Experience, tech tags, buttons
- **contact.css** - Contact grid, links, form inputs
- **footer.css** - Footer layout and links
- **animations.css** - All keyframes (fadeUp, spinSpark)
- **responsive.css** - Mobile breakpoints

### JavaScript Structure (4 Files)
- **cursor.js** - Mouse tracking, cursor styling, dark bg detection
- **animations.js** - Intersection Observer for scroll reveals & skill bars
- **form.js** - Contact form submission & feedback
- **navigation.js** - Active nav link highlighting on scroll

## Getting Started

1. Open `index.html` in your browser
2. Customize content directly in `index.html`
3. Modify styles in respective `css/` files
4. Add functionality in respective `js/` files

## Customization

### Colors
Edit CSS variables in `css/base.css`:
```css
:root {
  --lime: #7BF542;
  --purple: #C084FC;
  --orange: #FB923C;
  --pink: #F472B6;
  --sky: #38BDF8;
  --dark: #111111;
  --cream: #f0ede4;
  --white: #ffffff;
}
```

### Adding Features
- Add new CSS file in `css/` folder
- Add new JS file in `js/` folder  
- Link them in `index.html`
- Keep components isolated & modular

### Contact Information
Update email and social links in `index.html` contact section

### Resume
Replace `Resume.pdf` with your own resume file

## Advantages of This Structure

✅ **Modular** - Each component in its own file
✅ **Maintainable** - Easy to find and update specific features
✅ **Scalable** - Simple to add new sections
✅ **Clean** - Single index.html, organized folders
✅ **Performance** - Browser caches CSS/JS files separately
✅ **Flexibility** - Load/unload only what you need
✅ **Professional** - Industry-standard architecture

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers

## Future Enhancements

- Add dark mode toggle
- Connect to backend for email
- Add project portfolio section
- Blog or articles section
- Export resume as PDF
